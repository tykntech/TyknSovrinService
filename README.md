## Docker 
- Install docker

- Install docker compose

- run `docker-compose -f "docker-compose.yml" up -d --build`

## To run end2end tests

-  Install postman

- Import tests/credentialReqResp.postman_collection.json in postman

- Execute the rest api calls in postman
